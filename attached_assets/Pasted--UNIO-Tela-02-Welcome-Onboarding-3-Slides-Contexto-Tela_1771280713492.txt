# UNIO — Tela 02: Welcome Onboarding (3 Slides)

## Contexto
Tela de onboarding do app UNIO. Aparece após a Splash Screen. 
Três slides apresentam os módulos principais: Nutrição, Treino e Biometria.
O usuário pode navegar por swipe horizontal, dots ou botão "Próximo".

## Interações
1. Swipe horizontal entre slides (touch gesture)
2. Dots clicáveis para navegação direta
3. Botão "Próximo" avança slide (último slide: "Começar" navega para tela de Signup)
4. Botão "Pular" vai direto para Signup
5. Todas as transições de conteúdo: 0.4s ease com slide direction awareness
6. Cores se adaptam ao módulo do slide ativo (verde → laranja → azul)

## Animações
- Entrada do ícone: scale(0.8→1) + opacity(0→1), 0.4s cubic-bezier(0.34, 1.56, 0.64, 1)
- Troca de slide: translateX com direction awareness (left/right)
- Dot ativo: expand width 8→24px com spring curve
- Mudança de cor: todos os elementos coloridos transitam em 0.4s ease

## Acessibilidade
- Dots com aria-label "Slide 1 de 3"
- Botão "Pular" com aria-label "Pular introdução"
- Suporte a gestos de swipe (touch-action: pan-x)
- Contraste mínimo 4.5:1 em todos os textos
